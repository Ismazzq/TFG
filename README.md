TFG - Determinantes del gasto en seguros sanitarios privados

Repositorio local para versionado del TFG en LaTeX (Ismael Quesada García).

Estructura principal:
- `main.tex` - documento principal de LaTeX
- `capitulos/` - capítulos en .tex
- `figuras/` - imágenes y gráficos
- `datos_tfg_final.csv` - dataset final usado en estimaciones
- scripts Python para regenerar/analizar datos y generar figuras

Ramas recomendadas:
- `main` : versión estable / lista para compilar
- `dev` : desarrollo diario y cambios menores
- `experimental` : experimentos grandes, pruebas de modelos o cambios estructurales

Para conectar con un repositorio remoto (GitHub/GitLab), crea un repositorio remoto y ejecuta:

```bash
git remote add origin <URL-REMOTE>
git push -u origin main
```
